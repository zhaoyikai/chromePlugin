# 智能表单填写助手

一个Chrome浏览器扩展程序，可以自动识别支付页面并填写银行卡信息、持卡人姓名和账单地址。

## 功能特点

- ✅ 一键自动填写支付表单
- ✅ 自动选择银行卡支付方式  
- ✅ 智能识别表单字段
- ✅ 银行卡数据管理（最多100张卡片）
- ✅ 防重复使用（已使用的卡片自动标记）
- ✅ 随机生成美国人姓名
- ✅ 随机生成真实美国地址
- ✅ 多种触发方式（右键菜单、快捷键、扩展按钮）

## 安装方法

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展程序安装完成

## 使用方法

### 方法一：右键菜单（推荐）
1. 在任意支付页面右键点击
2. 选择"一键填写支付信息"
3. 扩展程序自动完成填写

### 方法二：键盘快捷键
1. 在支付页面按 `Ctrl + Shift + F`
2. 扩展程序自动完成填写

### 方法三：扩展按钮
1. 点击浏览器工具栏的扩展图标
2. 点击"立即填写当前页面"按钮

## 银行卡数据格式

银行卡数据格式为：`卡号|月|年|CVC`

示例：
```
5598880351374121|01|2026|641
5598880355787062|03|2030|314
5598880357867078|08|2027|498
```

## 数据管理

### 添加银行卡数据
1. 点击扩展图标打开管理界面
2. 在"银行卡数据管理"区域输入数据
3. 点击"更新银行卡数据"按钮

### 重置使用状态
1. 在管理界面点击"重置已使用的银行卡"
2. 所有银行卡重新变为可用状态

### 查看使用统计
- 总银行卡数量
- 已使用数量  
- 剩余可用数量

## 支持的网站

理论上支持所有基于HTML表单的支付页面，特别优化了：
- Stripe支付页面
- 标准HTML表单
- 大部分电商网站支付页面

## 填写内容

### 银行卡信息
- 银行卡号
- 到期时间（月/年）
- CVC安全码

### 持卡人信息
- 随机生成的美国人姓名

### 账单地址
- 随机真实美国地址
- 包含街道、城市、州、邮编

## 安全说明

- 所有数据仅存储在本地浏览器中
- 不会上传任何信息到服务器
- 银行卡数据仅用于测试目的
- 请勿使用真实银行卡信息

## 注意事项

1. 仅用于测试和开发目的
2. 请勿在生产环境使用真实支付信息
3. 使用前请确保了解相关法律法规
4. 建议在测试环境中使用

## 故障排除

### 扩展无法工作
- 确保扩展已正确安装和启用
- 刷新页面后重试
- 检查浏览器控制台是否有错误信息

### 填写不准确
- 某些网站可能使用特殊的表单结构
- 可以尝试多次操作
- 查看控制台日志了解详细信息

### 没有可用银行卡
- 在扩展管理界面添加更多银行卡数据
- 或者重置已使用的银行卡状态

## 技术实现

- **Manifest V3** Chrome扩展
- **Content Scripts** 页面内容操作
- **Background Service Worker** 数据管理
- **Chrome Storage API** 本地数据存储
- **Context Menus API** 右键菜单
